# Конспект по первому модулю
### CLI GIT

```bash
git checkout -b "type/name" # создает новую ветку

git status  #показывает статус репо

git add . #делает доступными для commit изменения

git commit -m "comment" #делает commit

git push #пушит коммит в репо

git push --set-upstream origin brancname #подключает локаль к уд. репо

git pull #синхронизируется с удаленным репо
```
